ubutun 数据库基操

<!--more-->



<font size=5> 

- 启动mysql：

```mysql
sudo service mysql start
```



- 停止mysql：

```mysql
sudo service mysql stop
```

- 重启mysql：

```mysql
sudo service mysql restart
```

- 查看mysql service 状态

```mysql
sudo service mysql status
```
